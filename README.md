# Dolar

Get the current conversion USD-ARS rate.

## Installation

```
> npm i -g dolar
```

## Usage

```
> dolar
```

Or if you prefer Macri to tell you the current rate:

```
> dolar macri
```

## Try without installing

```
> npx dolar
```
